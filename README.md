# bubble_sort

This is a Bubble sort project with Ruby. 
The project is focused on the use of yield and passing a block to a method. 

## Built With

- [ ] Ruby
- [ ] VS Code
- [ ] Ubuntu
- [ ] Rubocop linter

## Getting Started

To get your own copy of our project simply clone the repository to your local machine.

**Step 1**: Type the following command into a git shell

_git clone_ [_https://github.com/jadx2/bubble_sort.git_](https://github.com/jadx2/bubble_sort.git)

**Step 2**: Direct a terminal into the cloned repository directory

_cd bubble_sort_

**Step 3**: Run the program from command line, then open it in a text editor
